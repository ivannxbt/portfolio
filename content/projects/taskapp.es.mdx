## Descripción

Una aplicación moderna de gestión de tareas diseñada para equipos, con colaboración en tiempo real, funcionalidad de arrastrar y soltar, y seguimiento completo de proyectos.

## Características

- Tablero Kanban con arrastrar y soltar
- Actualizaciones en tiempo real via WebSocket
- Colaboración en equipo y permisos
- Comentarios y archivos adjuntos en tareas
- Fechas de vencimiento y recordatorios
- Análisis e informes de proyectos

## Stack Tecnológico

- React con TypeScript
- Backend Node.js Express
- Socket.io para funciones en tiempo real
- MongoDB para persistencia de datos
- Redis para caché

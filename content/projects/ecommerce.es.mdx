## Descripción

Esta plataforma de comercio electrónico proporciona una solución completa para tiendas en línea, incluyendo gestión de productos, funcionalidad de carrito de compras y procesamiento de pagos seguro.

## Características

- Catálogo de productos con búsqueda y filtrado
- Autenticación de usuarios y perfiles
- Carrito de compras con almacenamiento persistente
- Integración de pagos con Stripe
- Gestión de pedidos e historial
- Panel de administración para gestión de inventario

## Stack Tecnológico

Construido con tecnologías modernas:
- Next.js 14 con App Router
- TypeScript para seguridad de tipos
- PostgreSQL con Prisma ORM
- Stripe para pagos
- Tailwind CSS para estilos
